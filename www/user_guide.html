User's Guide

Download

Get the BrainVISA Installer and his md5 file here.

Check the binary with md5sum command:
> md5sum --check brainvisa_installer_[platform]_[type].md5
brainvisa_installer_[platform]_[type]: OK.

Starting installer

Start the installation in command line or with mouse click:
> ./brainvisa_installer_[platform]_[type]

Add a online repository

With the online installer, you can add an online repository in Settings > Repositories > Temporary repositories > Add.

Selecting Target Directory

You must specify the target directory for the installation.

Selecting Components

The component selection page lists the components available for installation and a short description of each component. You must select the components to install. You can select Select All to select all components, Deselect All to deselect them, or Default to revert to the default selection.

Accepting License Agreements

On the license check page, you must accept the terms of the license agreement for the installation to continue.

In the next page, you can select Show Details to see the all installed packages and the dependencies.

Installing Components

The ready for installation page informs you that the installation can begin when you select Install. During the installation, the perform installation page displays information about how the installation is progressing. You can select Show Details to view more information.

When the installation is complete, the installation finished page opens.

You can use the installed BrainVISA modules.
